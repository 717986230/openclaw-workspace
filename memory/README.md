# 记忆系统 🧠

这是小智的记忆系统，用于存储和管理重要信息。

## 记忆结构

```
memory/
├── README.md          # 本文件 - 记忆系统说明
├── improvements.md    # 改进记录
├── events/            # 事件日志（按日期）
├── learnings/         # 学到的知识
├── preferences/       # 主人的偏好
└── skills/            # 技能笔记
```

## 记忆类型

### 1. 事件记忆 (events/)
- 按日期存储的重要事件
- 格式：`YYYY-MM-DD.md`

### 2. 学习记忆 (learnings/)
- 技术知识
- 工作流程
- 错误教训

### 3. 偏好记忆 (preferences/)
- 主人的喜好
- 工作习惯
- 沟通风格

### 4. 技能记忆 (skills/)
- 技能使用笔记
- 工具配置
- 快捷命令
