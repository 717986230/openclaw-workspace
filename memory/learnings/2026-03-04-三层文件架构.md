# 2026-03-04 - 三层文件架构学习

## 📚 学习来源
Shubham Saboo 的文章 + Berryxia.AI 的中文翻译

## 🎯 核心洞见

### 1. 智能体不会因为用得久而变聪明 — 但文件会！
- 模型本身不变
- 变化的是：更丰富、更精准、更贴合需求的文件体系
- 这才是真正的护城河！

### 2. 三层文件架构
| 层级 | 核心问题 | 我们的文件 |
|------|----------|-----------|
| 身份层 | 这是谁？为谁服务？ | SOUL.md, IDENTITY.md, USER.md ✅ |
| 操作层 | 怎么干活？怎么自愈？ | AGENTS.md, HEARTBEAT.md ✅ |
| 知识层 | 学到了什么？ | MEMORY.md, 每日日志 ✅ |

### 3. 协作方式：文件就是集成层！
- 没有 API 调用
- 没有消息队列
- 只有文件系统
- 单写者原则：一个文件一个写者，多个读者

### 4. 进化的关键：反馈 + 记录
- 不是调 prompt
- 不是换模型
- 是说话，给反馈，看着它们把内容记下来
