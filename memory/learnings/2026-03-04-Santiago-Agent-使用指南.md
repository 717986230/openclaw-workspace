## 🛠️ Santiago 的专业玩法：Apify 平台

### 普通玩法的问题
- LLM 直接访问网页，会"阅读"HTML
- 因为 LLM 的"非确定性"，经常产生幻觉
- 返回的数据不准、格式混乱

### Santiago 的玩法：分工明确！
1. **Apify Actor（爬虫）**：用过去20年成熟的、确定性的爬虫工具去精准抓取原始数据
2. **LLM**：再用 LLM 去分析和处理这些干净、准确的数据

**一个抓取，一个分析，分工明确。这才是专业玩法，结果稳定、可靠。**

---

## 🚀 三步集成 Apify

### 第一步：安装 Apify 市场
在 Claude Code 的插件（Plugins）里，添加 Apify 官方的 GitHub 技能库，你就拥有了一个"爬虫应用商店"。

### 第二步：安装基础技能
比如安装"Lead Generation"插件。你可以直接让 Agent"帮我找3个我所在地区的厨房装修公司"，它会直接调用 Apify，返回一个带地址、电话、评分的精准表格。

**这已经秒杀了90%的普通搜索。**

### 第三步：解锁终极技能（真正的"10x"时刻）
安装"Actor Development"插件。

**这个插件的恐怖之处在于，它教会了 Agent 如何自己从零开始，为你构建一个全新的爬虫工具。**

Santiago 只用了10分钟，没写一行代码，就让 Agent 为他创建了一个"YouTube视频搜索爬虫"。

他想搜索任何主题的视频，Agent 就会自动运行这个部署在云端的爬虫，绕过反爬机制，把结果整理成列表返回给他。
