## 💡 今日关键提醒
1. 脑子里记的东西在会话重启后就消失了，文件不会
2. 当有人说"记住这个" → 更新记忆文件
3. 文字 > 大脑 📝
4. 不是调 prompt，不是换模型，不是重构架构 — 就是说话，给反馈，看着它们把内容记下来
5. 文件系统就是集成层，没有 API 调用，没有消息队列，只有 Markdown 文件
