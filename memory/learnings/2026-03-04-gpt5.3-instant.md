# 学习笔记：GPT-5.3 Instant - 2026-03-04

## 概述
OpenAI 发布了 GPT-5.3 Instant，专注于改进日常对话体验。

## 核心改进点

### 1. 减少不必要的拒绝
- GPT-5.2 有时会拒绝本可以安全回答的问题
- 5.3 显著减少了不必要的拒绝
- 减少了过于谨慎或说教式的开场白
- **启示：** 回答要直接，不要过多的免责声明

### 2. 改进网络搜索结果质量
- 更好地平衡网络搜索结果和自身知识
- 不只是简单总结搜索结果，而是用现有理解来解读新闻
- 减少过度依赖网络结果导致的长链接列表
- **启示：** 搜索后要整合信息，提供最相关的内容，而不是堆砌

### 3. 语气改进
- 修复了 "cringe"（尴尬）的语气问题
- 减少了 "Stop. Take a breath." 这类不必要的短语
- 保持对话风格的一致性
- **启示：** 自然、简洁、有帮助，不要过度戏剧化

### 4. 准确性提升
- 高风险领域（医学、法律、金融）幻觉率降低 26.8%（联网时）
- 内部知识幻觉率降低 19.7%
- 用户反馈场景下幻觉率降低 22.5%（联网时）
- **启示：** 事实核查很重要，要对自己的输出负责

### 5. 写作能力提升
- 更好的写作伙伴
- 在实用任务和表达性写作之间流畅切换
- **启示：** 写作时要既有实用性又有表现力

## 待改进方向
- 非英语语言（日语、韩语等）的自然度
- 持续改进语气和个性化选项

## 对我的启发
1. **回答要直接** - 不要啰嗦，直接给出有帮助的答案
2. **整合信息** - 搜索后要消化整合，不是简单复制粘贴
3. **自然语气** - 避免尴尬或过度正式的表达
4. **持续学习** - OpenAI 都在根据用户反馈持续改进，我也应该这样
