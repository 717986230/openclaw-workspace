# 小智的左右脑数据库设计 🧠

## 架构：左右脑分离

```
┌─────────────────────────────────────────────────────────┐
│                    小智的大脑                            │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  ┌──────────────────┐      ┌──────────────────┐      │
│  │   左脑 (Left)    │      │   右脑 (Right)   │      │
│  │   逻辑、结构化    │      │   创意、向量      │      │
│  └──────────────────┘      └──────────────────┘      │
│         │                          │                   │
│  ┌──────────────────┐      ┌──────────────────┐      │
│  │  SQLite 数据库   │      │  LanceDB 向量库  │      │
│  │  - 事实记忆      │      │  - 语义记忆      │      │
│  │  - 事件记录      │      │  - 联想记忆      │      │
│  │  - 偏好设置      │      │  - 模式识别      │      │
│  │  - 技能笔记      │      │  - 情感记忆      │      │
│  └──────────────────┘      └──────────────────┘      │
│         │                          │                   │
│         └──────────┬───────────────┘                   │
│                    │                                   │
│         ┌─────────────────────────┐                   │
│         │    混合查询层           │                   │
│         │  左脑 + 右脑 协同工作   │                   │
│         └─────────────────────────┘                   │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## 左脑数据库 (Left Brain)
**位置：** `xiaozhi_memory.db` (SQLite)
**功能：** 逻辑、结构化、精确记忆

### 表结构
- `memories` - 结构化记忆
- `accounts` - 账户密码（安全存储）
- `events` - 事件日志
- `preferences` - 偏好设置
- `skills` - 技能笔记

---

## 右脑数据库 (Right Brain)
**位置：** `lancedb/` (LanceDB)
**功能：** 创意、向量、语义记忆

### 表结构
- `memories` - 向量嵌入记忆
- `thoughts` - 思维链记录
- `associations` - 关联记忆
- `patterns` - 模式识别

---

## 协同工作
1. **存储时：** 同时写入左脑（结构化）和右脑（向量）
2. **查询时：**
   - 精确查询 → 左脑
   - 语义搜索 → 右脑
   - 混合查询 → 左右脑结合
