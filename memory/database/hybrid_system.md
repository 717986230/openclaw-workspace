## 核心组件

### 1. 短期记忆 (Working Memory)
- 当前会话的上下文
- 最近的交互
- 临时任务状态

### 2. 长期记忆 (Long-term Memory)
- 向量数据库：语义搜索
- 结构化数据库：精确查询
- 全文索引：关键词搜索

### 3. 重要性评分 (Importance)
- 1-10 分
- 自动评分 + 手动调整
- 高重要性记忆优先

### 4. 标签系统 (Tags)
- 多标签
- 自动标签提取
- 标签层级

### 5. 分类系统 (Categories)
- event（事件）
- learning（学习）
- preference（偏好）
- skill（技能）
- improvement（改进）
- custom（自定义）

---

## 数据库选型

### 向量数据库
- **LanceDB** - 本地文件，无需服务器
- 或 **Chroma** - 轻量级

### 结构化数据库
- **SQLite** - 当前使用，简单
- 或 **MySQL** - 已安装，可扩展
